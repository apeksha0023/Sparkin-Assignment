
# 📁Dynamic Dashboard with React & Chakra UI
This project is a modern, responsive, and interactive dashboard built using React.js and Chakra UI. It features a clean UI with dark mode support, real-time updates, and various components designed for managing accounts, tracking activities, and optimizing cost-saving actions.

## Live Demo🚀


<a href="https://sparkin-assignment.vercel.app/" style="color: rgb(211, 28, 255);">Demo</a>

## Features 📌

 ✅**User Dashboard**
   - Displays account details including Name, Type, and ID.
   - Shows insights like total and active accounts.
   - Provides a list of recent activities for quick tracking.
  
  ✅**Cost-Saving Actions**
   - Allows users to manage AWS resources efficiently, such as:
   - Start/Stop Resources (EC2, RDS, Light Sail, Amazon Neptune).
   - Pause Reserve Resources (Redshift Clusters, Aurora Serverless V2).
   - Resource Cleanup (Terminate EC2, Delete EBS Volumes, Snapshots).
   - Built using Checkbox selections for ease of use.

   ✅**JAM Policy Viewer**
   - Displays JSON-based policy details with a copy-to-clipboard feature.
   
  ✅**Navigation Bar(Sticky)**
   - Back, Forward, and Refresh buttons for easy navigation.
   - Dark Mode Toggle using Chakra UI’s theme system.
   - Stays sticky on scroll for accessibility.

   ✅**Fully Responsive UI**
    - Works seamlessly on desktop, tablet, and mobile devices.
    - Implements Chakra UI’s flexible grid and layout system.


    
## Tech Stack⚙️
- Frontend: React.js (Vite), Chakra UI
- State Management: React Hooks
- Icons: React Icons & Chakra UI Icons
- Routing: React Router DOM




**Deployment:** Vercel


## Installation🛠️

Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository
```
Install dependencies 
```bash
npm install
```
Run the development server
```bash
npm run dev
```
## Deployment
- The project can be easily deployed on Vercel, Netlify, or GitHub Pages.
- Use npm run build to create a production-ready build.
